プライバシーポリシー

最終更新日: 2026年3月16日

NexusDiary — AI日記アプリ（Android）

本アプリは、ユーザーのプライバシー保護を最優先に設計されています。収集する情報は サービス提供に必要な最低限にとどめ、第三者への販売は一切行いません。

第1条 収集する情報

（1）Googleアカウント情報（Google Sign-In 時のみ）

取得項目氏名・メールアドレス・プロフィール画像
利用目的ユーザー識別・日記データのクラウド同期
保存先Firebase Authentication（Google LLC）

（2）日記データ

取得項目日記テキスト・投稿日時・AIタグ・感情タグ・デバイス情報
利用目的日記の表示・AI分析・クラウド同期
保存先Firebase Firestore（Google LLC）。ユーザー自身のみがアクセス可能。

（3）AI 分析用テキスト

取得項目日記テキストの一部
利用目的タグ・感情・開発アイデアの自動生成（Gemini API 使用）
処理方式Firebase Functions サーバー上でのみ処理。APIキーはクライアントに保存されません。

（4）広告識別子（Android）

取得項目広告識別子（Google Advertising ID）
利用目的Google AdMob によるパーソナライズ広告配信
オプトアウトAndroid 設定「広告」→「広告のパーソナライズを無効化」から変更できます

第2条 外部サービスへの情報提供

本アプリは以下の外部サービスを利用します。各サービスのプライバシーポリシーを必ずご確認ください。

  • Firebase Authentication / Firestore（Google LLC）
    認証・データ保存に使用します。 Google プライバシーポリシー
  • Firebase Functions（Google LLC）
    Gemini API 呼び出しをサーバーサイドで処理します。クライアントから直接 AI API へは接続しません。
  • Gemini API（Google LLC）
    日記テキストを AI 分析するために Firebase Functions 経由で利用します。 Gemini API 利用規約
  • Google AdMob（Google LLC）
    広告配信のために利用します。広告識別子を取得・使用します。 Google プライバシーポリシー

上記以外の第三者への情報提供、および法令に基づく開示要請への対応以外で、ユーザー情報を第三者に販売・共有することはありません。

第3条 データの保護

  • 日記データは Firestore セキュリティルールにより、ユーザー自身のみがアクセス可能です（request.auth.uid == userId を検証）。
  • Gemini API との通信は Firebase Functions サーバーを経由します。APIキーはクライアントアプリに保存されません。
  • 通信はすべて HTTPS により暗号化されます。
  • 本アプリは端末内への機密情報（APIキー等）の永続保存を行いません。

第4条 データの削除

ユーザーは以下の方法でデータを削除できます。

  • アプリ設定画面の「アカウントを削除」から、Firestore 上のすべての日記データおよびアカウント情報を削除できます。
  • 削除後のデータ復元はできません。操作の前にご確認ください。
  • 削除を希望する場合はお問い合わせフォームからもご依頼いただけます（第6条参照）。

第5条 ポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。 重要な変更がある場合は、アプリ内またはこのページで通知します。 定期的にご確認いただくことをお勧めします。

変更後のポリシーは、このページに掲載した時点で効力を生じます。
